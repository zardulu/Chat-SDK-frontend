# Chat SDK Frontend
Made using Svelte + Tailwind

Link - https://silly-jelly-56fe39.netlify.app/

# Installation
1. Clone the repository:
```bash   
git clone https://github.com/your-username/chat-app.git
cd chat-app
```
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```
